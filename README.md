# 🌤️ Weather App

A weather app that shows current weather and 5-day forecast for any city.


## Features

- Current weather (temperature, humidity, wind, feels like)
- 5-day forecast
- Interactive map
- Toggle °C / °F
- Dark / Light mode
- Share weather info
- Last city saved

## Technologies

- HTML, CSS, JavaScript
- OpenWeatherMap API
- Leaflet.js

## How to Run

1. Clone the repo
2. Get your free API key from [OpenWeatherMap](https://openweathermap.org/)
3. Replace `apiKey` in `script.js`
4. Open `index.html`
